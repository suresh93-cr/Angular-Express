Angular is a TypeScript-based free and open-source web application framework
led by the Angular Team at Google and by a community of individuals and corporations.
Angular is a complete rewrite from the same team that built AngularJS.


React is a free and open-source front-end JavaScript library for building user interfaces
based on UI components. It is maintained by Meta and a community of
individual developers and companies.



Vue.js is an open-source model–view–viewmodel front end
JavaScript framework for building user interfaces
and single-page applications.
